# Select ImageViewer

This project is a small image viewer

## Used Tools

- React
- Redux
- Axios
- React Router
- TypeScript

## How can run this project

1. Please clone project
```
git clone https://github.com/htadayoni/image-viewer.git
```

2. Run yarn install for install packages
```
yarn install
```

2. Run project by yarn
```
yarn start
```
